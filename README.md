### KissanAI

An AI-powered agricultural decision support system for farmers in Punjab.

Features:

- crop recommendation intelligence
- profitability prediction
- weather-risk analysis
- irrigation optimization
- smart land allocation
- real-time weather integration
- interactive analytics dashboards

- FastAPI
- React
- TypeScript
- PostgreSQL
- Tailwind CSS

### ExplainableIDS

An explainable machine learning framework for network intrusion detection.

Core ideas:

- intrusion detection using machine learning
- Random Forest and XGBoost models
- SHAP-based global explanations
- LIME-based local explanations
- interpretable security dashboard
- explainable AI for cybersecurity decision support
